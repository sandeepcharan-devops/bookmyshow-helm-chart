# 🚀 Helm Installation Steps

## 1️⃣ Create Helm Chart

```bash
helm create BookMyShow-chart
```

---

## 2️⃣ Rename Chart to Lowercase

Kubernetes resource names must be lowercase.

```bash
mv BookMyShow-chart bookmyshow-chart
cd bookmyshow-chart
```

---

## 3️⃣ Update Chart.yaml

### Before

```yaml
name: BookMyShow-chart
```

### After

```yaml
name: bookmyshow-chart
```

---

# ⚙️ values.yaml Configuration

```yaml
replicaCount: 1

image:
  repository: yourdockerhub/bookmyshow
  pullPolicy: IfNotPresent
  tag: latest

service:
  type: ClusterIP
  port: 80

ingress:
  enabled: true
  className: traefik

  hosts:
    - host: ""
      paths:
        - path: /
          pathType: Prefix

autoscaling:
  enabled: false

serviceAccount:
  create: false
  name: ""
```

# 🔥 Errors Faced and Solutions

---

## ❌ Error 1

```text
invalid release name
```

### Cause

Helm release names must be lowercase.

### Wrong

```bash
helm template BookMyShow .
```

### Correct

```bash
helm template bookmyshow .
```

---

## ❌ Error 2

```text
.Values.serviceAccount.create nil pointer evaluating interface
```

### Cause

Missing serviceAccount section in values.yaml.

### Solution

```yaml
serviceAccount:
  create: false
  name: ""
```

---

## ❌ Error 3

```text
.Values.autoscaling.enabled nil pointer evaluating interface
```

### Cause

autoscaling section missing.

### Solution

```yaml
autoscaling:
  enabled: false
```

---

## ❌ Error 4

```text
.Values.httpRoute.enabled nil pointer evaluating interface
```

### Cause

Unused HTTPRoute template generated by helm create.

### Solution

Remove template:

```bash
rm templates/httproute.yaml
```

---

## ❌ Error 5

```text
cannot re-use a name that is still in use
```

### Cause

Failed Helm release still existed.

### Solution

```bash
helm uninstall bookmyshow
```

Then reinstall:

```bash
helm install bookmyshow .
```

---

## ❌ Error 6

```text
src refspec main does not match any
```

### Cause

No commit existed or branch mismatch.

### Solution

```bash
git add .
git commit -m "Added Helm chart"
git branch -M main
git push -u origin main
```
